Color Wheel Sort

Built with p5.js

A unique visual representation of a variety of common sorting algorithms using a color wheel. Each slice in the circle represents a hue in the color spectrum, which are mapped to a digit between 0 and 359. These can then be shuffled and sorted to show how various sorting algorithms work in real time.

Mosaic Quilt

Built with JavaScript.

Implementation of various image processing algorithms from scratch. Includes basic pixel operations like brightness/contrast control, as well as convolution operations such as sharpening, blurring, edge detection, and dithering. Apply a quilt-like mosaic mask given a set of album and film covers.

DMV Auto Booker

Built with Python and Selenium.

This Selenium-based webscraper automates the process of booking a DMV appointment for drivers license/state ID/learners permit renewals and road tests. These can take several months to book due to high demand, as openings are sparse and can be snagged within a few minutes of freeing up. This script checks various locations and searches intermitently until a valid appointment is found.
